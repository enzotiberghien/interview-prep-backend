# InterviewPrep Backend

This is the backend of the InterviewPrep application, designed to help users prepare for job interviews by generating interview questions and providing AI feedback.

## Technologies Used

- .NET Core (ASP.NET Core)
- Entity Framework Core
- PostgreSQL
- Docker
- Azure
- OpenAI API

## Features

- User authentication
- Upload and store job descriptions
- Generate interview questions using OpenAI API
- Store user responses
- Provide AI feedback on
